## 技术栈

React Next Typescript TailwindCss FramerMotion googleMap 

## 页面介绍

1.首页 包括注册 以及 footer
2.Success page 注册成功后会跳转
3.Contact Us Page：可以提交留言以及查看google map地图

## 主要特性

- 使用 Tailwind CSS 
- 完美适配移动端、平板和桌面端设备

- Framer Motion 实现动画

- 遵循 WCAG 无障碍标准
- 支持键盘导航
- 语义化 HTML 结构
- ARIA 标签支持

- 客户端表单验证
- 实时输入验证反馈
- 错误提示信息
- Mock API 响应处理

- Google Maps 集成


## 安装和运行

### 安装依赖

```bash
pnpm install
```

或使用 npm:

```bash
npm install
```

### 开发模式

```bash
pnpm dev
```

或:

```bash
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000) 

### 构建生产版本

```bash
pnpm build
```

### 启动生产服务器

```bash
pnpm start
```
